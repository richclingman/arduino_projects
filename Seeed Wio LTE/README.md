* Sketch

* Use **SerialUSB** for for output (NOT Serial)


#### Upload Firmware
* Hold *Boot* button
* Press and release *Reset* button
* Release *Boot* button


#### Run Firmware
* Press and release *Reset* button
* Monitor output on *Tera Term*
